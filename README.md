# 🌡️ Kerala Heat Watch · കേരള ചൂട് നിരീക്ഷണം

A real-time heat and weather monitoring dashboard for Kerala, aggregating official data from the India Meteorological Department (IMD) and Kerala State Disaster Management Authority (KSDMA).

**Live site:** https://medonthomas-alt.github.io/Kerala_Heat_Watch

---

## What this does

- Shows live temperature, humidity, and rainfall for 14+ Kerala weather stations
- Displays official KSDMA heat alerts in Malayalam with colour-coded severity
- Provides 7-day forecasts for all stations
- Embeds live UV index data from KSDMA IoT sensor network (14 stations across Kerala)
- Shows KSDMA official temperature and humidity maps
- Publishes public advisories from KSDMA in a readable format
- Updates automatically every 4 hours via GitHub Actions

## How It Works

```
Every 4 hours:
GitHub Actions robot
  → fetches IMD CD_Status_Forecast.json    (forecast stations)
  → fetches IMD Observed_Stations.json     (AWS stations)
  → fetches IMD HWCW_Status.json           (warning colours)
  → scrapes sdma.kerala.gov.in/temperature (KSDMA alert text + maps + issued time)
  → filters Kerala stations (official codes + geographic boundary)
  → detects alert colour from Malayalam text (yellow/orange/red)
  → saves combined data/weather.json
  → commits to repo

Website (index.html):
  → reads data/weather.json from same repo
  → renders station cards, map, alerts, forecasts
  → embeds live KSDMA UV IoT dashboard (updates every 30 seconds)
  → auto-refreshes every 30 minutes
```

## Technical Stack

- **Frontend:** Vanilla HTML + CSS + JavaScript (no framework)
- **Map:** Leaflet.js v1.9.4 with CartoDB dark tiles
- **District GeoJSON:** Kerala Open Maps project
- **Automation:** GitHub Actions (4-hour cron)
- **Backend fetch:** Python 3.10 with requests + BeautifulSoup4
- **Hosting:** GitHub Pages (free)
- **UV data:** ICFOSS OpenIoT Grafana dashboard (embedded iframe)

## Data Reliability Disclaimer

This dashboard aggregates publicly available data from IMD and KSDMA for informational purposes only. Data may have delays or gaps depending on source availability. Station readings represent specific geographic points and may not reflect conditions across an entire district.

**For official warnings and accurate forecasts, always refer to:**
- [sdma.kerala.gov.in](https://sdma.kerala.gov.in) — Kerala SDMA
- [mausam.imd.gov.in/thiruvananthapuram](https://mausam.imd.gov.in/thiruvananthapuram/) — IMD Thiruvananthapuram
